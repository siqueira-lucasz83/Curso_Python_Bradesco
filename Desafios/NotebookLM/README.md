# README — Caderno Temático no NotebookLM

## Descrição
Este projeto consiste na criação de um caderno temático no NotebookLM voltado ao estudo introdutório
de um tema financeiro, utilizando fontes abertas em texto e PDF como base de conhecimento.

## Objetivo
Utilizar a IA como ferramenta de aprendizagem ativa, organizando informações, formulando perguntas
estratégicas e produzindo um material de estudo estruturado e reutilizável.

## Metodologia
- Seleção de 3 a 5 fontes confiáveis (artigos, textos ou PDFs).
- Definição de objetivos de estudo claros.
- Elaboração e teste de prompts variados no NotebookLM.
- Registro das respostas geradas pela IA, com referência às fontes.
- Curadoria crítica do conteúdo.

## Resultado Esperado
- Miniguia de estudo com resumos organizados.
- Glossário de conceitos fundamentais.
- Conjunto de prompts reutilizáveis para revisões futuras.
- Material estruturado para apoiar o aprendizado contínuo.

## Observação
A atividade enfatiza pensamento crítico, organização do conhecimento e uso consciente da IA como
apoio ao estudo, não como substituta da análise humana.
